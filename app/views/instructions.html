¡Bienvenido a CONEMO!

CONEMO es un programa que te permite leer textos, mirar vídeos y responder preguntas, todo con el objetivo de mejorar tu salud física y emocional.

Esta sesión de entrenamiento contiene explicaciones y demostraciones de cómo usar cada una de las funciones de CONEMO. Te mostraremos cómo

  • Seleccionar “siguiente y atrás”
  • Ver vídeos
  • Elegir respuestas
  • Elegir varias respuestas
  • Elegir una sola respuesta
  • Retroceder
  • Abrir diálogos
  • Ver las notificaciones de sesiones
  • Usar el calendario
  • Programar una hora

Siguiente y atrás

Cuando abres el aplicativo y la sesión que corresponde de ese día, lee el texto de esa página y luego presiona el botón “siguiente”

Si quieres regresar a la página anterior presiona el botón “atrás”.

Ver vídeos

Para ver un vídeo, presiona el botón “agrandar pantalla” y voltea el teléfono en posición horizontal.

Luego, presiona el triángulo en el centro

Para pausar el vídeo, presiona el botón “pausa”

Para regresar al aplicativo cuando termine el vídeo, presiona la pantalla para ver los botones del teléfono y presiona el botón “atrás”

video4

Elegir respuestas

Para responder una pregunta o elegir actividades, simplemente presiona el rectángulo blanco.

Se abrirá una página con todas las opciones.

Para elegir una opción presiónala.

Una vez que hayas hecho tu selección, presiona “Aceptar”.

Elegir varias respuestas

En algunas preguntas se puede elegir varias opciones. Para elegir cualquiera de esas opciones presiónala.

Si has elegido una respuesta y te gustaría cambiarla, presiona nuevamente la respuesta seleccionada para “des-marcarlo”. Acá un ejemplo:

¿Qué tipo de comida te gusta? Elige todas las opciones que corresponden.

Elegir una sola respuesta

En otras preguntas sólo debes elegir una opción de las presentes. Acá un ejemplo:

¿Cuál es tu color favorito?

Retroceder

Recuerda, siempre puedes apretar el botón “atrás” en tu teléfono si no quieres responder una pregunta en ese momento o si quieres regresar a la sesión.

Para volver a la página inicial de CONEMO, presiona el símbolo de la casita.

Diálogos

A veces recibirás un mensaje del sistema, un “diálogo”, al cual deberías responder.

Ese aparecerá en tu pantalla y te indicará qué es lo que deberías hacer.

Después de que hayas seleccionado una de las opciones, una respuesta aparecerá en tu pantalla.

Presiona “OK” cuando terminas de leerlo.

Notificaciones de sesiones

Además te mandaremos notificaciones para recordarte que hay una nueva sesión disponible en CONEMO que debes realizar.

Esa notificación aparecerá en forma de un símbolo en la esquina derecha de tu pantalla.

Desliza tu dedo encima de la pantalla desde el borde superior hacia abajo, para ver la notificación.

Luego presiona el mensaje de notificación para abrir el aplicativo CONEMO.

Calendario

Para seleccionar una fecha en la cual quieres hacer una actividad, presiona el rectángulo blanco correspondiente.

Se abrirá un calendario en el cual puedes cambiar el día, mes y año. Presiona el “+” para aumentar el número o mes y “-” para disminuirlo. Luego presiona “Establecer”

Hora

Para programar una hora en la cual quieres hacer la actividad elegida, presiona el rectángulo blanco y selecciona la hora correspondiente.

Para modificar la hora, presiona el “+” para cambiar la hora hacia más adelante y “-” para cambiarlo hacia atrás. Si quieres cambiar “AM” o “PM”, presiona el botón “AM” o “PM”.
